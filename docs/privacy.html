Snippet Mini プライバシーポリシー

最終更新日: 2026年7月7日

Snippet Mini（以下「本アプリ」）は、開発者 Yamaguchi Shohei（以下「開発者」）が提供する macOS アプリケーションです。 本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明するものです。

1. 収集する情報

本アプリは、ユーザーの個人情報を収集しません。 ユーザーが登録したスニペット（定型文）は、端末内のアプリ専用フォルダにのみ保存され、開発者のサーバーへ送信されることはありません。

2. 第三者への提供

本アプリは、ユーザーのデータを第三者と共有しません。

3. ネットワーク通信

本アプリは、通常の利用においてインターネット通信を行いません。

4. トラッキング

本アプリは、広告識別子の利用やユーザー行動のトラッキングを行いません。

5. データの保存場所

スニペットデータは、Mac 上の Application Support フォルダ内に JSON 形式で保存されます。 アプリを削除すると、当該データも削除されます（macOS の仕様に従います）。

6. お問い合わせ

本ポリシーに関するお問い合わせは、GitHub の Issues よりご連絡ください。
